# 每日股研AI

**AI量化算法即時評估A/H股。**

每日股研AI是一款面向單只股票研究的分析產品。用戶輸入股票名稱或代碼後，系統會聚合行情、估值、板塊、資訊、財務和量化指標，輸出簡潔、可執行、可解釋的分析結果。

當前 MVP 聚焦 A 股與 H 股，優先做好移動端可讀性、即時分析頁、動態估值區間、六維健康評分與點位計劃。

## 產品定位

本產品不是傳統資訊聚合頁，也不是通用聊天機器人。它要回答的是：

- 當前價格是否合理
- 動態估值區間在哪裡
- 成長邏輯是否有產業、護城河和兌現證據支撐
- 關聯板塊是否配合
- 最新資訊偏利好、利空還是中性
- 關注位、確認位、失效位分別在哪裡

## 核心模組

| 模組 | 說明 |
| --- | --- |
| 首頁搜尋 | 支援 A/H 股名稱、代碼和別名搜尋 |
| 即時分析頁 | 單只股票的結論、估值、評分、板塊、資訊和點位 |
| 六維健康評分 | 價值、估值性價比、成長、盈利能力、財務、分紅 |
| 動態估值區間 | 由價格、歷史行情、波動率、財務和風險因素生成 |
| 點位計劃 | 關注區、確認位、失效位 |
| 關聯板塊 | 相關度與即時板塊漲跌 |
| 精選資訊 | 最新動態資訊，標記利好、利空、中性 |
| 行業趨勢 | 用量化方向分表達行業趨勢影響 |

## 評分原則

六維健康評分要求可解釋、可校準、不誇張。成長分不因熱門概念無條件提高，而是看完整證據鏈：

```text
主營業務 -> 行業賽道 -> 產業鏈位置 -> 護城河 -> 商業兌現
```

ST、持續虧損、現金流惡化、審計風險、退市風險和財務惡化會壓低相關評分。

```text
static          前端構建產物
docs            項目文檔
```

## 本地運行

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

cd apps/dsa-web
npm install
cd ../..
```

啟動後端：

```bash
PYTHONPATH=. ./.venv/bin/python -m uvicorn server:app --host 0.0.0.0 --port 8000
```

啟動前端：

```bash
npm --prefix apps/dsa-web run dev -- --host 0.0.0.0
```

## 構建與測試

```bash
npm --prefix apps/dsa-web run build
PYTHONPATH=. ./.venv/bin/python -m compileall api tests
```

## 部署說明

部署到中國內地雲服務器時，建議優先使用國內可訪問的數據和模型服務。真實 API Key 應通過環境變量注入，不要提交到 Git 歷史。

## 風險說明

本項目處於 MVP 階段，分析結果用於研究和產品展示，不構成投資建議。
